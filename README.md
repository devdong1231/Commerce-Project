### Main.java
 - 시작 지점이 되는 클래스

### Cart.java
 - 장바구니 클래스
 - 장바구니 상품 추가 / 삭제 기능
 - 구매 기능

### CartItem.java
 - 장바구니에 담길 아이템(product와 amount(구매할 수량))을 field로 가짐

### Category.java
 - 카테고리 이름, 상품 정보를 List의 형태로 가지고 있는 class

### CommerceSystem.java
 - 프로그램 비즈니스 로직 클래스
 - 장바구니 관리
 - 재고 관리

### Product.java
 - 개별 상품 정보를 가지는 클래스
 - 상품명, 가격, 설명, 재고수량


### Cumstomer.java
 - 고객명, 이메일, 등급을 가지고 있는 클래스


### IOHandler.java
 - 입출력을 관리하는 클래스
